# Memory Directory

This directory contains persistent data and progress tracking files for the OBLIRIM project.

## Purpose

- Store application state
- Track configuration changes
- Log system information
- Keep user preferences
- Progress tracking for installations and updates

## Files

- `system_info.json` - Cached system information
- `config.json` - Application configuration
- `install_progress.log` - Installation progress tracking
- `user_settings.json` - User preferences and settings

## Usage

This directory is automatically created during installation and should not be deleted.
The application will recreate necessary files as needed.
